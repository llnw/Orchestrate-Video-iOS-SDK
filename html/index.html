    Limelight Video Platform iOS SDK

    In line with Limelight’s mission to provide a consistent feature set across online and device playback, including analytics, closed captioning, DRM, ad monetization, etc, this iOS SDK enables a full player experience while imposing minimal integration requirements on Orchestrate Video customers.

    Basic Usage / Overview

    General steps for using the SDK:

    • Create an instance of LVKLibrary with your Limelight Orchestrate Video API credentials
    • Retrieve an LVKMedia from your LVKLibrary instance
    • Display an LVKPlayerController or LVKPlayerViewController instance and assign the media to it.
    • Hit play! :beers:

    Note, this is very similar to how you would use Apple’s MPMediaPlayerController or MPMediaPlayerViewController.

    Hello World

    In a new iOS project, in AppDelegate.m:

    @implementation AppDelegate

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions
{
  // 1) Initialize a library instance with your Limelight Video API credentials
  LVKLibrary *library = [ LVKLibrary libraryWithOrganizationId:@"[org id]"
                                                    accessKey:@"[access key]"
                                                       secret:@"[secret]" ];

  // 2) Set up an LVKPlayerViewController to use in the media success callback
  LVKPlayerViewController *playerViewController = [LVKPlayerViewController new]

  // 3) Fetch a media. There are many ways to do this, and we recommend using
  //    our Reactive Cocoa API methods vs the plain callback style shown here.
  //    See full docs for more info.
  [library
    fetchMediaForId:@"[ your media id ]"
    success: ^(LVKMedia *aMedia) {

      // 4) Profit!
      playerViewController.media = aMedia;
      [playerViewController play];
    }
    failure: ^(NSError *error) {
       NSLog(@"Media fetch error: %@", error);
    }];

  // stock window setup; using the playerViewController directly for this example
  self.window = [[UIWindow alloc] initWithFrame:[[UIScreen mainScreen] bounds]];
  self.window.rootViewController = playerViewController;
  [self.window makeKeyAndVisible];
}

@end

    Components For Advanced Uses

    • LVKPlayerController: Play video with your own view management.
    • LVKAnalyticsReporter: Implement analytics in a custom video app.
    • LVKDRMDecoder: Play back DRM-enabled content in a custom video app.

    Installation

    Prerequisites

    1. A Limelight Video Platform account. You can sign-up for a free, 30-day trial.
    2. A Macintosh computer with Xcode. You can install Xcode from the Mac App Store.

    Installation

    CocoaPods (preferred)

    1. Set up CocoaPods

    2. Add LimelightVideoKit to your Podfile:

      pod 'LimelightVideoKit'

    3. Execute $pod install

    Manual

    1. [Download] the static library to your computer.
    2. Add the static library to your Xcode project. To add a static library to an Xcode project, drag and drop the file onto Xcode.
